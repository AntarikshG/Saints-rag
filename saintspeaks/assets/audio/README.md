# Audio Assets

Place your 10-second background audio file here with the name `quote_background.mp3`.

The audio file will be used when exporting Quote of the Day as MP4 video.

Recommended audio file specifications:
- Format: MP3, M4A, or WAV
- Duration: 10 seconds
- Sample Rate: 44.1 kHz
- Bit Rate: 128 kbps or higher
